◆作品について◆

◇使用言語 : C++
◇SDK      : DirectX9
◇開発期間 : 2022/12～2023/02(3ヶ月)
◇ジャンル : 3Dアクション

◇作品説明
・1対1のボス戦で、攻撃を繋げたり強攻撃を放ったりしてボスを倒す速度を競うゲームです。

◇工夫した点
・プレイヤーが敵から離れている場合でも「何もできない時間」を作らないようにしました。
・プレイヤーの状態とキー押下などのアクションを使用し、硬直を無視して連続攻撃を繰り出せるようにしました。
・強攻撃を実装したことでコンボを溜め、維持するメリットを増やしました。
・プレイヤーとの距離によって敵が行動を変え、攻撃を当てに来るようになっています。
  { player.cpp / player.h } { enemy.cpp / enemy.h }

◇反省点
・プレイヤーの攻撃パターンが少なく、攻撃の順番を工夫させることが出来ませんでした。
・剣の軌跡以外のエフェクトを用意することが出来ず、攻撃を繋げている感が不足していました。

◆操作方法◆
「キーボード」
・WASDキー		→	プレイヤーの移動
・QEYBキー 	 	→	カメラの\旋回
・Uキー			→	通常攻撃(続けて押すことで2種類の攻撃があります)
・Iキー			→	回転切り
・Oキー			→	弾の発射(押している間発射し続けます)
・Pキー 		→	回避(WASDと併用で方向指定)
・左SHIFTキー	→	攻撃強化(コンボ30以上の時にBYと併用で強攻撃)
・SPACEキー 	→	ロックオン状態の切り替え

「ジョイパッド」
・左スティック	→	プレイヤーの移動
・右スティック	→	カメラの旋回
・Bボタン		→	通常攻撃(続けて押すことで2種類の攻撃があります)
・Yボタン		→	回転切り
・Aボタン 		→	回避(WASDと併用で方向指定)
・LBボタン		→	弾の発射(押している間発射し続けます)
・RBボタン		→	攻撃強化(コンボ30以上の時にBYと併用で強攻撃)
・右スティック押し込み →	ロックオン状態の切り替え
